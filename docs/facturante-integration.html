Facturante

Como funciona la integracion de facturacion electronica con AFIP en bigg-finance

1 Que es Facturante

Facturante es el servicio externo que se encarga de emitir comprobantes electronicos ante AFIP. bigg-finance le envia los datos de la factura o nota de credito, Facturante lo procesa con AFIP, y nos devuelve el numero oficial y el PDF del comprobante.

2 Como estan conectadas las partes

El flujo pasa por 4 capas

Pantalla del usuario

bigg-finance (navegador)

El usuario completa el formulario de facturacion: elige la franquicia, tipo de comprobante, monto, fecha. Puede emitir una factura individual o muchas a la vez (masiva).

Facturador

Crear factura o nota de credito individual

Pendientes

Ver facturas que todavia no se emitieron

Detalle franquicia

Ver estado de cuenta y descargar PDFs

Envia los datos de la factura
Servidor (Vercel)

Funcion intermedia

Recibe los datos del navegador, los transforma al formato que necesita Facturante, agrega las credenciales de autenticacion (que nunca salen del servidor por seguridad) y hace la llamada.

Envia el comprobante armado
Facturante

Servicio de facturacion

Recibe el comprobante, lo valida, y lo presenta ante AFIP para su autorizacion. Tambien genera el PDF oficial del comprobante.

Solicita autorizacion
AFIP

Administracion Federal de Ingresos Publicos

Autoriza el comprobante y le asigna un numero oficial unico (ej: FA 0100-00000010). Este numero es el que aparece en la factura impresa.

3 Flujo de emision de una factura

Que pasa desde que el usuario hace click hasta que tiene su factura

1

El usuario completa el formulario

Elige: tipo de comprobante (Factura o Nota de Credito), franquicia, fecha, monto neto, y la cuenta a la que aplica (Fee, Pauta, etc).

2

Se verifica si corresponde emitir por Facturante

Solo se emite electronicamente si la franquicia es de Argentina, la moneda es pesos (ARS) y la empresa emisora es NAKO SRL. Si no cumple, se asigna un numero manual.

3

Se arma el comprobante con todos los datos

Se incluye: datos del emisor y receptor (CUIT, razon social, condicion IVA), tipo de comprobante, montos con o sin IVA, periodo del servicio, condicion de pago.

4

Se envia a Facturante

Facturante recibe el comprobante, lo valida y lo presenta ante AFIP. Si hay algun dato incorrecto, devuelve un error con el detalle del problema.

5

AFIP procesa y asigna el numero

AFIP puede tardar unos segundos en procesar. El sistema consulta repetidamente (hasta 8 veces) hasta obtener el numero oficial asignado.

6

Se guarda el resultado

Se almacena el numero de factura (ej: "FA 0100-00000010") y el ID interno de Facturante (necesario para descargar el PDF oficial despues).

7

Se descarga el PDF automaticamente

Si esta disponible, se baja el PDF oficial de AFIP. Si todavia no esta listo, se genera un HTML como alternativa temporal.

5 Como funciona una Nota de Credito

Las notas de credito siempre hacen referencia a una factura emitida previamente

1

Se selecciona la factura original

El usuario elige de un listado cual es la factura que quiere anular o ajustar. Solo aparecen facturas ya emitidas con numero AFIP.

2

Se completa el monto a acreditar

Puede ser el total de la factura (anulacion completa) o un monto parcial (ajuste).

3

Se emite con referencia al comprobante original

La nota de credito incluye el numero y fecha de la factura original. AFIP requiere esta referencia para validarla. Si la factura era tipo A, la NC es NCA; si era B, es NCB.

4

Siempre se emite como "contado"

Independientemente de como se haya cobrado la factura original, las notas de credito siempre se registran con condicion de pago contado.

8 Descarga del PDF oficial

Como se obtiene el comprobante en PDF despues de emitido

1

Automatico despues de emitir

Apenas se emite una factura, el sistema intenta descargar el PDF oficial de AFIP automaticamente.

2

Manual desde el detalle de franquicia

En cualquier momento se puede hacer click en el numero de factura para volver a descargar el PDF.

3

Se consulta a Facturante

El servidor pide a Facturante la URL del PDF generado y lo descarga.

4

Alternativa si no esta listo

Si AFIP todavia no proceso el comprobante, se puede descargar una version en HTML generada localmente como alternativa temporal.

10 Que pasa si algo falla

Cada etapa tiene su propia estrategia de manejo de errores

SituacionReintenta?Que pasa
Facturante rechaza el comprobante No Se muestra el error de AFIP al usuario para que corrija los datos
AFIP tarda en asignar el numero Si, 8 veces Consulta cada 2.5 segundos. Si no llega, la factura queda emitida pero sin numero visible todavia
El PDF no esta disponible No Se ofrece descargar una version HTML alternativa. El PDF se puede reintentar despues
Falla una factura en emision masiva No Se registra el error y se continua con la siguiente franquicia
